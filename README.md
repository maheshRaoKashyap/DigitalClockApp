Description

This is a desktop application built using Java and Swing, which displays a real-time digital clock. The clock features dark red text on a light-colored background, making it clear and visually appealing.

The project demonstrates core Java programming, GUI development, and layout management using GridLayout. The clock updates every second using a thread-safe Swing Timer, showing proficiency in handling real-time updates in a desktop application.

Developed in IntelliJ IDEA, this project also showcases experience with font styling, color management, and component alignment. All code is version-controlled with Git and hosted on GitHub, reflecting standard software development practices.

Tech Stack / Tools Used

Language: Java 11+

GUI Framework: Swing

IDE: IntelliJ IDEA

Version Control: Git & GitHub

Key Concepts: GridLayout, JLabel, Font & Color styling, Swing Timer, Thread-safe UI updates

Features

Real-time digital clock with 1-second updates

Dark red text for high visibility on light background

Simple, clean, and professional UI

Demonstrates Java Swing GUI development and layout management

Thread-safe updates using Swing Timer
